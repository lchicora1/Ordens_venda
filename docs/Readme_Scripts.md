OVGS API - README

Visão Geral
API REST desenvolvida em NestJS para gerenciamento de pedidos, clientes, transportadoras, produtos, agendamentos e auditoria.

Como executar com Docker Compose
•	docker compose down -v
•	docker compose up --build

API: http://localhost:3000
Prisma Studio: http://localhost:5555


Scripts úteis
•	npm install
•	npm run start:dev
•	npm test
•	npm run test:cov
•	npx prisma generate
•	npx prisma migrate dev
